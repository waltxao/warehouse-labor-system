# v2.2.0 设计文档：Apple HIG 重设计 + 页面介绍 + 企业微信推送

## 概述
三项功能变更：
1. 全量重写前端设计系统为 Apple HIG 风格
2. 每个页面底部添加始终可见的使用说明区
3. 新增企业微信推送配置页面

## 需求1：Apple HIG 设计系统

### 字体
- 移除 Google Fonts Fira Sans/Code
- 改用 -apple-system 系统字体栈
- 等宽：SF Mono

### 配色
- 主色 #007AFF, 强调色 #FF9500, 背景 #F2F2F7
- 侧边栏纯黑 #1C1C1E，无毛玻璃

### 圆角/阴影/动效
- 卡片16px, 按钮12px, 弹窗20px
- Spring cubic-bezier(0.34,1.56,0.64,1)
- 卡片hover scale(1.02)

### 涉及文件
- main.css + 9个Vue视图 + 3个组件

## 需求2：页面介绍 + 系统信息

### PageIntro 组件
- 可复用组件，放在每页底部
- 浅灰背景，圆角16px

### 各页面说明
- 总览/对比/上传/AI/告警/推送/用户/设置 各有专属说明

### 全局版本信息
- AppLayout底部：v2.2.0 · 作者：Walt · © 2026

## 需求3：企业微信推送

### 数据模型
- webhook_configs 表：warehouse_id, webhook_url, notify_users

### API
- CRUD + /webhooks/push

### 推送服务
- 图片消息 base64 + 文字消息含 @mention
- 文字："以上{start}-{end} {warehouse}倉實際出勤人數與實際需求人力情況，請留意"

### 前端
- NotificationConfigView.vue 配置页
- DashboardView 推送按钮（选单仓时显示）
- AppLayout 侧边栏新增菜单项
